# task2_Rune
### 能量机关识别<br>
#### 主要思路<br>
先二值化并膨胀，防止受流动条干扰<br>
之后根据轮廓外接旋转矩形的长宽比和面积比，锁定待打击的扇叶（相比其他扇叶轮廓面积较小）<br>
再对内层轮廓外接旋转矩形，得到装甲板，旋转矩形对角线连线的中点为打击点<br>
对于圆心R的识别，除了限制长宽比外，还直接限制了轮廓面积<br>

#### 使用方法
(在项目文件夹下) <br>
```
mkdir build
cd build
cmake ..
make
cd ../bin
./main
```
